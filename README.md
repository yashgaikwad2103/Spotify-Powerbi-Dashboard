# Spotify Data Analytics Dashboard

## Project Overview

This project is an interactive Spotify Data Analytics Dashboard built using Microsoft Power BI.

The dashboard provides insights into songs, artists, albums, popularity trends, and music statistics using interactive visualizations.

---

## Features

- KPI Cards
  - Distinct Songs
  - Artist Count
  - Average Duration
  - Average Popularity

- Artist Analysis

- Song Popularity Analysis

- Album Type Analysis

- Explicit vs Non-Explicit Songs

- Monthly Popularity Trends

- Year-wise Analysis

- Interactive Filters

---

## Dashboard Pages

### 1. Overview

- Overall KPIs
- Popularity Trend
- Album Analysis
- Songs by Artist
- Distinct Songs by Month

### 2. Artist Dashboard

- Songs by Artist
- Popular Songs
- Artist Hit Analysis
- Song Details Table
